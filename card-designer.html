Business Card Designer

I created a business card designer as a way to practice JavaScript. This project was solely created using HTML, CSS, and JavaScript. I used Figma to refine my ideas for the page. The end result is a form that will be filled out by the user which will instantly display the input information showing the user a preview of the card they created.
