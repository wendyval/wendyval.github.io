Guess the Number Game

I created this fun project, Guess the Number game, to practice JavaScript. To create this project, I used HTML, CSS, and JavaScript. To come up with the design concept for the game, I made a couple of mockup designs using Figma. In the end, I was able to create a guessing game where the user needs to guess the computer-generated number in under a limited number of attempts.
